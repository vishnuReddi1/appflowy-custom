# AppFlowy Custom

A customized version of **AppFlowy**, built for learning, experimentation, and extending the original open-source workspace with additional features.

> **Based on the original AppFlowy project by the AppFlowy team.**

---

## Overview

This repository contains my customized version of AppFlowy. It is intended for exploring modern cross-platform application development using Flutter and Rust while experimenting with productivity, collaboration, and AI-powered workspace features.

## Installation

Clone the repository:

```bash
git clone https://github.com/vishnuReddi1/appflowy-custom.git
```

Move into the project:

```bash
cd appflowy-custom
```

Follow the official AppFlowy development setup instructions for your operating system.
